Staxel - Lesser Than Better Start Mod

A quality-of-life mod for the game Staxel, created by Gaming on Insulin.

About the Mod

This mod is designed to enhance the early game experience in Staxel by adjusting various settings and adding new features to help players get started more smoothly. It's perfect for both new and returning players who want to jump into the action without the initial grind.
